# 考途 StudyAgent｜Notion 作品集文案

## 一句话介绍

面向考研、考公和教材型考试复习场景，基于 Dify Workflow 搭建的备考教材重点梳理与个性化复习智能体。

## 我的角色

独立完成产品定位、用户痛点拆解、Dify Workflow 搭建、Prompt 设计、测试复盘和作品集整理。

## 项目背景

备考学生面对教材时经常遇到三个问题：重点难判断、笔记整理慢、复习计划不具体。考途 StudyAgent 尝试将教材复习流程拆解为多个 AI 节点，帮助用户从章节内容生成完整学习报告。

## 核心流程

```text
用户输入 → 章节结构解析 → 重点知识提炼 → 背诵笔记生成 → 自测题生成 → 答案与解析生成 → 复习计划生成 → 报告输出
```

## 为什么选择 Workflow

相比普通 Chatbot，Workflow 更适合处理教材复习这种连续任务。每个节点只负责一个明确步骤，输出更稳定，也更方便测试和迭代。

## 当前限制

- 需要恢复模型额度或配置自有模型 API 才能稳定公开演示。
- 当前未接入考试大纲和真题知识库。
- 长文本教材仍需要按章节或小节分段处理。

## 下一步计划

1. 接入考试大纲和真题知识库。
2. 增加错题复盘节点。
3. 增加学习进度追踪。
4. 完成更多真实教材测试。
