# 中文简历项目描述

## 版本一：完整版本

考途 StudyAgent｜备考教材重点梳理与个性化复习智能体  
AI 产品项目｜Dify Workflow / Prompt Engineering / EdTech Agent

- 面向考试备考场景，独立设计并基于 Dify Workflow 搭建教材重点梳理智能体，完成从教材输入到复习方案输出的完整产品流程。
- 将复杂学习任务拆分为章节解析、重点提炼、背诵笔记、自测题、答案解析和复习计划等多个 LLM 节点，提升输出结构的稳定性与可解释性。
- 设计各节点 Prompt、输入输出规范及节点依赖关系，并针对长文本、无效输入、模型幻觉和输出一致性制定测试方案。
- 输出 PRD、AI Workflow 设计、Prompt 文档、测试报告及项目复盘，并通过 GitHub 与个人作品集完成项目展示。

## 版本二：压缩版本

- 基于 Dify Workflow 搭建备考教材学习 Agent，将教材复习拆分为章节解析、重点提炼、背诵笔记、自测题、答案解析和复习计划等节点。
- 设计结构化 Prompt 与输入输出规范，提升学习报告生成的稳定性、可解释性和可复用性。
- 输出 PRD、Prompt 文档、测试报告和作品集材料，用于展示 AI 产品从需求定义到原型验证的完整流程。
