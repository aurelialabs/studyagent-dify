# 中文简历项目描述

考途 StudyAgent｜备考教材重难点梳理与个性化复习智能体  
AI 产品项目｜Dify Workflow / Prompt Engineering / EdTech Agent

- 独立设计并基于 Dify Workflow 搭建备考教材学习 Agent，将教材输入拆解为章节解析、重点提炼、背诵笔记、自测题、答案解析和复习计划等多节点流程。
- 设计各节点 Prompt、输入输出规范和变量依赖关系，通过结构化 Markdown 输出提升学习报告的稳定性、可读性和可复用性。
- 围绕长文本、无效输入、扫描版 PDF、模型幻觉和答案题目错位等风险，输出测试方案、迭代记录和后续优化规划。
- 完成 PRD、AI Workflow 设计、Prompt 文档、测试报告、GitHub 项目和作品集文案，形成可演示、可复盘的 AI 产品项目材料。
