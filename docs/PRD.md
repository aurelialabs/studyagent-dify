# 考途 StudyAgent PRD

## 1. 产品定位

考途 StudyAgent 是一个面向考试备考场景的教材学习 Agent，帮助用户基于教材章节内容生成结构化复习报告，覆盖章节解析、重点提炼、背诵笔记、自测题、答案解析和复习计划。

## 2. 目标用户

- 考研学生
- 考公及事业单位备考用户
- 教资、法考、职业资格考试备考用户
- 需要复习教材型课程的大学生

## 3. 用户痛点

1. 教材长，重点难判断。
2. 笔记整理耗时，质量不稳定。
3. 自测题和答案解析需要额外查找。
4. 复习计划不具体，缺少执行路径。

## 4. MVP 范围

### 本期包含

- 用户输入学习信息
- 章节结构解析
- 重点知识提炼
- 背诵笔记生成
- 自测题生成
- 答案与解析生成
- 复习计划生成
- 最终学习报告输出

### 本期不包含

- 用户账号体系
- 长期学习进度追踪
- 整本教材自动解析
- 真题知识库检索
- 自动批改主观题
- 扫描版 PDF 自动 OCR
- Word 文档自动导出

## 5. 输入字段

| 字段 | 说明 |
|---|---|
| exam_type | 考试类型 |
| subject | 科目名称 |
| book_name | 教材或书名 |
| chapter_title | 章节标题 |
| chapter_text | 教材内容/章节正文 |
| current_level | 当前基础 |
| daily_time | 每日可用时间 |
| review_period | 复习周期 |

## 6. 输出模块

1. 章节结构
2. 核心重点
3. 背诵笔记
4. 自测题
5. 答案解析
6. 复习计划

## 7. 验收标准

| 模块 | 验收标准 |
|---|---|
| 章节结构 | 不新增原文不存在的章节层级，能说明概念关系 |
| 重点知识 | 区分必背、理解、易混淆和可能考查内容 |
| 背诵笔记 | 结构清晰，可直接用于背诵 |
| 自测题 | 覆盖重点，有题型和难度，不提前泄露答案 |
| 答案解析 | 严格对应题目，说明依据和知识点 |
| 复习计划 | 符合用户基础、内容量和每日时间 |

## 8. 风险与限制

- 模型可能基于常识补充输入中没有的内容。
- 长文本输入可能超出模型上下文。
- Dify Cloud 免费额度可能影响演示稳定性。
- 未接入真题知识库时，考查方式为模型推断，不能替代官方大纲。
- 扫描版 PDF 本质上是图片，文档提取可能失败，需要先 OCR 成可复制文本。
- 直接生成 Word 文件需要额外接入导出插件或外部文档生成服务。
