# 产品设计说明

## 产品定位

考途 StudyAgent 是面向教材型备考场景的 AI 学习助手，重点解决“教材内容到复习行动”的转化问题。

## 设计原则

1. 先结构化，再生成内容。
2. 每个节点只承担一个明确任务。
3. 输出要能直接用于复习。
4. 用户输入尽量表单化，减少提问成本。
5. 对未验证和信息不足的内容保持提示。

## 输入设计

输入字段分为三类：

- 学习材料：文件上传、章节正文、章节标题、学习范围。
- 个性化信息：当前基础、每日可用时间、复习周期。
- 考试上下文：考试类型、科目、教材名称。

## 输出设计

输出固定为六个模块：

1. 章节结构
2. 核心重点
3. 背诵笔记
4. 自测题
5. 答案与解析
6. 复习计划

## 交互设计

当前使用 Dify Web App 作为 MVP 交互界面。前端不是当前必须项，后续如需要增强作品集展示，可做独立 Landing Page 或学习报告展示页。

## 版本规划

- v1.0：基础 Workflow + Web App Demo。
- v1.1：文件上传和文档提取优化。
- v1.2：异常输入处理。
- v2.0：OCR 和长文本分段。
- v2.1：Word 导出和知识库增强。
