## 4. 测试关注点

- 章节层级是否准确。
- 重点是否覆盖材料关键内容。
- 自测题是否与重点匹配。
- 答案是否逐题对应。
- 复习计划是否可执行。
- 是否编造输入中不存在的信息。
- 相同输入多次运行输出结构是否稳定。
