# 竞品分析

## 对比对象

| 产品形态 | 优点 | 局限 |
|---|---|---|
| 普通 ChatGPT 问答 | 灵活，输入门槛低 | 输出依赖提问质量，流程不稳定 |
| 传统学习 App | 内容体系完整 | 个性化生成能力较弱 |
| AI 笔记工具 | 擅长总结材料 | 不一定覆盖自测和复习计划 |
| Dify Workflow Agent | 流程可控，节点清晰 | 需要配置模型与 Prompt |

## 差异化定位

考途 StudyAgent 不做泛问答，而是围绕教材复习构建固定学习闭环：

```text
教材输入 → 结构解析 → 重点提炼 → 背诵笔记 → 自测题 → 答案解析 → 复习计划
```

## 为什么选择 Workflow

Workflow 适合将复杂学习任务拆成多个可调试节点。相比普通 Chatbot，它更容易保证输出结构稳定，也更方便定位某一步的问题。
